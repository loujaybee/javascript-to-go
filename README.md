
# JavaScript To Go

A compilation of syntax similarities / differences between JavaScript and Go.

### Exercise: For Loop

__Prints the numbers 1-10.__

* In Go: `go run ./for_loop`
* In NodeJS: `node ./for_loop`

### Exercise: Map

__Maps an array__

* In Go: `go run ./map`
* In NodeJS: `node ./map`

### Exercise: Filter

__Filters an array__

* In Go: `go run ./filter`
* In NodeJS: `node ./filter`
